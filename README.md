Sentiment Analysis

Description

This project uses Machine Learning to classify reviews as Positive, Negative, or Neutral.

Technologies Used

- Python
- Pandas
- NLTK
- Scikit-Learn

Files

- reviews.csv
- sentiment_analysis.py
- requirements.txt

How to Run

1. Install libraries:
   pip install -r requirements.txt

2. Run the program:
   python sentiment_analysis.py

3. Enter a review and view the predicted sentiment.

Example

Input:
I love this phone

Output:
Sentiment: positive
